Cloudflare Pages — обязательные настройки (Workers & Pages → ваш проект → Settings → Builds)

1) Build command (НЕ "npm run build"):
   npm run cf:build

2) Build output directory:
   .vercel/output/static

3) Deploy command — ОЧИСТИТЬ полностью (удалить "npx wrangler deploy").
   Для Pages с Git деплой идёт автоматически после сборки.
   Команда "wrangler deploy" только для Workers и ломает Pages.

4) Переменная NODE_VERSION = 20 (опционально).

5) Settings → Functions → compatibility flag: nodejs_compat

После сохранения — Retry deployment.
